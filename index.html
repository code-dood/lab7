Program 1: Creation of Block

AIM

To create a block using hashing in blockchain.

ALGORITHM

  1. Define Block class
  2. Initialize index, timestamp, previous hash, and data
  3. Generate hash using SHA-256
  4. Display block details

PROGRAM


import hashlib
import time

class Block:
    def __init__(self, index, previous_hash, data):
        self.index = index
        self.timestamp = int(time.time())
        self.previous_hash = previous_hash
        self.data = data
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        text = f"{self.index}{self.timestamp}{self.previous_hash}{self.data}"
        return hashlib.sha256(text.encode()).hexdigest()

block = Block(1, "0", "This is my first block")

print("Index:", block.index)
print("Hash:", block.hash)
